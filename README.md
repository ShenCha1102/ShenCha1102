## About me 👋

我是焚天烈渊寂灭恐惧战马，需要三个乡村乐转职达成棱彩大羁绊



首先我学习了**初始化操作**（init）：
1. 点击VScode最左侧的第三个tab“源代码管理”，或快捷键ctrl+shift+G；
2. 点击“初始化存储库”。

刚刚我学会了**拉取操作**（PULL）：
1. 在VScode中按F1或 ctrl+shift+P；
2. 输入git:clone；
3. 再输入需要拉取的项目网址；
4. 后续可以在“图表”工具栏，点击倒数第三个按钮“拉取”。

**配置“.gitignore文件”**：
1. 在项目文件夹里创建一个名为.gitignore的md文件；
2. 逐行加入不需要上传git的相对路径。

完成配置后，我尝试了**暂存操作**：
1. 先ctrl + S保存文件（只有保存了之后才可以在下文的“更改”处看到更改的文件信息”；
2. 打开“源代码管理”；
3. 在“更改”下悬停刚刚更改的文件，点击“+”；
4. 文件被加入“暂存的更改”。

又学习了**提交操作**：
1. 打开“源代码管理”；
2. 输入需要提交的信息；
3. 点击“√提交”。

如果需要上传到Github或Gitlab，就需要用到**推送操作**：
1. 打开“源代码管理”；
2. 点击下方“图表”工具栏的倒数第二个按钮“推送”。

学习了**创建新分支**：
1. 找到并点击左下角的第三个图表（形如“源代码管理”图表），显示的是“当前分支名”；
2. 点击上方“创建新分支”；
3. 自动切换到该分支。

当重复拉取后或创建新分支后，可以进行**分支切换**：
1. 打开“源代码管理”；
2. 点击下方“图表”工具栏的第一个按钮“分支操作”；
3. 在上方选择需要切换的分支；
4. 也可以点击左下角第三个图表，显示的是“当前分支名”，然后在上方选择分支。




**ctrl + ~可以打开集成终端**
